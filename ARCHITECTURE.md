# CORTEX 架构设计文档

## 1. 设计哲学

### 1.1 核心假设

CORTEX 基于以下跨学科假设：

1. **意识作为信息整合与广播的相变**：借鉴 Global Workspace Theory (GWT/GNW)，意识不是单一脑区的产物，而是信息达到临界阈值后在全局网络中广播的涌现属性。
2. **树突作为独立计算单元**：生物神经元的树突分支具有局部非线性计算能力，而非简单的加权求和。
3. **脉冲编码提供高效稀疏通信**：大脑使用脉冲（spike）进行通信，这种事件驱动机制提供了极高的能量效率和稀疏性。
4. **预测是认知的核心**：大脑本质上是预测机器，通过最小化预测误差来学习和感知。
5. **多时间尺度是记忆的基础**：从毫秒级的突触可塑性到分钟级的神经调节，多时间尺度的动态交互构成了记忆的神经基础。

### 1.2 与现有架构的对比

| 特性 | Transformer | Mamba/SSM | Spiking NN | CORTEX (本架构) |
|------|------------|-----------|------------|-----------------|
| 注意力复杂度 | O(N²) | O(N) | O(N) | O(N log N) |
| 生物合理性 | 低 | 中 | 高 | 高 |
| 稀疏激活 | 无 | 部分 | 完全 | 混合 |
| 全局信息广播 | 通过注意力 | 状态压缩 | 无显式机制 | GNW 点火机制 |
| 树突计算 | 无 | 无 | 部分 | 完整 DCU |
| 预测编码 | 无 | 无 | 无 | 集成 |
| 多时间尺度 | 无 | 部分 | 部分 | 异质耦合 |
| 意识状态建模 | 无 | 无 | 无 | 显式 workspace |

## 2. 全局架构

```
输入嵌入 → [LPM₁, LPM₂, ..., LPMₙ] → Global Workspace → 输出投影
                ↑___________________________↓
                        (反馈连接)
```

### 2.1 局部处理模块 (Local Processing Module, LPM)

每个 LPM 模拟大脑的一个功能特化区域（如视觉皮层、语言区域等）。结构上包含：

- **Dendritic Computation Layer**: 树突计算层
- **Multi-Timescale State Layer**: 多时间尺度状态层
- **Predictive Coding Layer**: 预测编码层

### 2.2 全局工作空间 (Global Neuronal Workspace, GNW)

GNW 是整个架构的核心创新，实现以下功能：

1. **信息竞争 (Competition)**: 各 LPM 通过竞争性选择机制争取进入工作空间
2. **阈值点火 (Ignition)**: 当某信息的显著性超过阈值时触发全局广播
3. **全局广播 (Broadcasting)**: 点火后的信息被广播到所有 LPM
4. **意识状态向量 (Consciousness State)**: 输出当前"意识内容"的低维表示

## 3. 核心组件数学描述

### 3.1 树突计算单元 (DCU)

生物神经元接收数千个突触输入，这些输入在树突分支上被非线性地局部整合。DCU 数学建模如下：

#### 3.1.1 树突分支计算

对于第 i 个神经元的第 b 个树突分支：

```
h_{i,b} = σ(W_{i,b} · x + b_{i,b})
```

其中：
- `x ∈ R^d` 是输入向量（前突触输入）
- `W_{i,b} ∈ R^{d_branch × d}` 是分支权重
- `σ` 是分支非线性（通常为 ReLU 或带阈值的 sigmoid）
- `b` 是分支索引

#### 3.1.2 分支类型

每个 DCU 包含多种分支类型，模拟生物树突的功能分化：

- **兴奋性分支** (`exc_ratio` 比例): 使用 ReLU/SiLU 非线性
- **抑制性分支** (`inh_ratio` 比例): 使用负斜率非线性
- **整流分支**: 使用 Softplus 模拟 NMDA 受体的电压门控特性
- **调制分支**: 接受神经调质输入（如多巴胺、乙酰胆碱）

#### 3.1.3 胞体整合

```
s_i = Σ_b α_{i,b} · h_{i,b}      (带权聚合)
y_i = σ_soma(s_i + β · max_b h_{i,b})   (胞体非线性)
```

其中 `α_{i,b}` 是可学习的分支权重，`β` 控制最大值门控的强度。

#### 3.1.4 脉冲输出

```
spike_i = 1 if y_i > θ_i else 0
```

使用替代梯度（surrogate gradient）进行反向传播：

```
∂spike/∂y ≈ 1 / (1 + |y - θ|²)   (三角形替代梯度)
```

### 3.2 选择性状态空间与脉冲门控

在传统 SSM（如 Mamba）基础上引入脉冲稀疏门控：

#### 3.2.1 连续时间状态方程

```
h'(t) = A · h(t) + B · x(t) · g(t)
y(t) = C · h(t)
```

其中 `g(t)` 是脉冲门控信号：

```
g(t) = spike(t) ⊙ σ_gate(x(t))
```

`spike(t)` 由 DCU 产生，`σ_gate` 是连续门控。

#### 3.2.2 离散化

使用零阶保持 (ZOH) 离散化：

```
Ā = exp(A · Δ)
B̄ = A⁻¹ · (Ā - I) · B

h_k = Ā · h_{k-1} + B̄ · x_k · g_k
y_k = C_k · h_k
```

其中 `Δ` 是数据相关的离散化步长（Mamba 的选择机制）。

#### 3.2.3 选择性参数

B 和 C 是输入依赖的：

```
B_k = Linear_B(x_k)
C_k = Linear_C(x_k)
Δ_k = softplus(Linear_Δ(x_k) + Δ_offset)
```

### 3.3 全局工作空间层

#### 3.3.1 显著性计算

每个 LPM m 在时间步 t 输出显著性分数：

```
s_{m,t} = ||z_{m,t}||₂ · γ_m   (L2范数 + 可学习增益)
```

`z_{m,t}` 是 LPM 的输出，`γ_m` 是模块特定的显著性增益。

#### 3.3.2 竞争性选择

使用带温度参数的 softmax 竞争：

```
p_{m,t} = exp(s_{m,t} / τ) / Σ_j exp(s_{j,t} / τ)
```

其中 `τ` 是温度。当 `τ → 0` 时退化为 hard max（winner-take-all）。

#### 3.3.3 点火阈值与广播

全局工作空间状态：

```
W_t = Σ_m p_{m,t} · z_{m,t}    (加权聚合)

ignition_t = σ(W_t · w_ignition + b_ignition)
```

当 `ignition_t > θ_ignition` 时：
1. 触发全局广播
2. 更新意识状态向量 `C_t`
3. 强化与当前内容相关的连接（短期可塑性）

#### 3.3.4 意识状态向量

```
C_t = (1 - α) · C_{t-1} + α · W_t · ignition_t
```

`C_t` 是一个低维向量（如 256 维），可视为模型的"当前意识内容"。

### 3.4 集成预测编码

#### 3.4.1 预测-误差分解

对于 LPM 的每一层 l：

```
μ_l = f_pred(h_{l-1})      # 对下一层的预测
ε_l = h_l - μ_l            # 预测误差

h_l' = h_l - λ · ε_l       # 误差修正后的表示
```

其中 `f_pred` 是预测函数（通常为小 MLP），`λ` 是学习率。

#### 3.4.2 Precision 加权

不同误差的可靠性不同：

```
π_l = σ(Linear_π(h_{l-1}))   # precision (逆方差)
ε_l^{weighted} = π_l ⊙ ε_l
```

#### 3.4.3 损失函数

```
L_pred = Σ_l ||ε_l||²_{π_l} = Σ_l Σ_i π_{l,i} · ε_{l,i}²
```

### 3.5 多时间尺度异质处理

#### 3.5.1 时间尺度定义

每个 LPM 包含多个时间尺度的子网络：

```
τ_fast = 10ms    (对应 γ ~ 40Hz 振荡)
τ_medium = 100ms (对应 α ~ 10Hz 振荡)
τ_slow = 1000ms  (对应 δ ~ 1-3Hz 振荡)
```

#### 3.5.2 多时间尺度状态更新

```
τ · dh_τ/dt = -h_τ + f(x, h_other_τ)
```

离散化：

```
h_τ^{(t)} = (1 - dt/τ) · h_τ^{(t-1)} + (dt/τ) · f(x_t, {h_τ'^{(t-1)}})
```

#### 3.5.3 跨尺度耦合

```
h_{combined} = concat([h_{τ_1}, h_{τ_2}, ..., h_{τ_n}])
y = W_cross · h_{combined}
```

### 3.6 意识门控注意力

#### 3.6.1 注意力分数

```
Q = Linear_Q(x)
K = Linear_K(x)
V = Linear_V(x)

# 基础内容注意力
A_content = softmax(Q · K^T / √d)

# 意识调制
A_conscious = A_content ⊙ (C_t · W_attn + b_attn)

# 输出
y = A_conscious · V
```

## 4. 训练方法

### 4.1 多目标损失

```
L_total = L_task + λ₁ · L_pred + λ₂ · L_consciousness + λ₃ · L_sparse
```

其中：
- `L_task`: 任务损失（交叉熵、MSE 等）
- `L_pred`: 预测编码损失
- `L_consciousness`: 意识状态一致性损失（时序平滑）
- `L_sparse`: 脉冲稀疏性损失（鼓励低发放率）

### 4.2 替代梯度训练

对于脉冲神经元，前向传播使用阈值函数，反向传播使用替代梯度：

```
# 前向
spike = heaviside(v_mem - θ)

# 反向
∂spike/∂v_mem ≈ surrogate(v_mem - θ)
```

### 4.3 课程学习

1. 阶段1：训练连续部分（关闭脉冲门控）
2. 阶段2：渐进引入脉冲稀疏性
3. 阶段3：完整联合训练（含工作空间竞争）

## 6. 复杂度分析

### 6.1 时间复杂度

对于序列长度 N：

- 标准自注意力: O(N² · d)
- Mamba SSM: O(N · d)
- **CORTEX GNW**: O(N · d · log M)，其中 M 是 LPM 数量
- **CORTEX DCU**: O(N · d · B)，其中 B 是分支数

总体: **O(N · d · (B + log M))** ≈ 线性于序列长度

### 6.2 空间复杂度

- KV Cache (Transformer): O(N · d)
- SSM 状态: O(d)
- **CORTEX 状态**: O(d · (1 + T))，T 是时间尺度数

### 6.3 计算稀疏性

由于脉冲门控，实际活跃计算量约为理论值的 10-30%（取决于阈值设定）。
